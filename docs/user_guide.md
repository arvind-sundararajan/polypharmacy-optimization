# User Guide
## Introduction
The polypharmacy-optimization project aims to optimize personalized medication regimens for geriatric patients with polypharmacy.
## Getting Started
1. Clone the repository: `git clone https://github.com/your-username/polypharmacy-optimization.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
## Using the Application
1. Load the medication database: `python load_database.py`
2. Optimize medication regimens: `python optimize_regimens.py`
## Troubleshooting
* Check the logs for errors: `tail -f logs/error.log`
* Contact support: [support@example.com](mailto:support@example.com)
